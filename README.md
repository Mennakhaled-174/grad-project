# Graduation Project Backend

A scalable backend application developed using NestJS and TypeScript as part of our Graduation Project.

## Features

- User Authentication & Authorization
- Login & Registration System
- Real-Time Chat Functionality
- AI Integration
- Course Management System
- E-Commerce Features
- User Profile Management
- RESTful APIs
- Database Integration

## Technologies Used

- NestJS
- Node.js
- TypeScript
- MongoDB
- JWT Authentication
- Socket.IO

## Installation

```bash
npm install
```

## Run the Project

```bash
npm run start:dev
```

## Team Project

This project was developed as part of our Graduation Project to provide a complete platform with AI-powered features, communication tools, course management, and e-commerce functionality.
